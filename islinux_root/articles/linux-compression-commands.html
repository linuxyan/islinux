📦 Essential Linux Compression and Archive Commands

10 min read

Introduction

Compression and archiving are essential tasks for backup, file transfer, and storage optimization. Linux provides powerful command-line tools for creating, extracting, and managing compressed archives. This guide covers 10 essential commands for compression and archive management.

1. tar - Tape Archive

Create and extract archive files (often combined with compression).

# Create uncompressed archive
tar -cvf archive.tar file1 file2

# Extract archive
tar -xvf archive.tar

# List archive contents
tar -tvf archive.tar

# Create gzip compressed archive
tar -czvf archive.tar.gz directory/

# Extract gzip archive
tar -xzvf archive.tar.gz

# Create bzip2 compressed archive
tar -cjvf archive.tar.bz2 directory/

# Extract bzip2 archive
tar -xjvf archive.tar.bz2

# Create xz compressed archive
tar -cJvf archive.tar.xz directory/

# Extract to specific directory
tar -xzvf archive.tar.gz -C /destination/

Options: -c=create, -x=extract, -t=list, -v=verbose, -f=filename, -z=gzip, -j=bzip2, -J=xz

2. gzip - GNU Zip Compression

Compress and decompress files using gzip algorithm.

# Compress file (replaces original)
gzip file.txt

# Compress keeping original
gzip -k file.txt

# Decompress
gzip -d file.txt.gz

# Set compression level (1-9)
gzip -9 file.txt        # Maximum compression
gzip -1 file.txt        # Fastest compression

# Show compression ratio
gzip -l file.txt.gz

# Recursive compression
gzip -r directory/

3. gunzip - Decompress gzip Files

Decompress files compressed with gzip.

# Decompress file
gunzip file.txt.gz

# Decompress keeping original
gunzip -k file.txt.gz

# Decompress to stdout
gunzip -c file.txt.gz > file.txt

# Test integrity
gunzip -t file.txt.gz

Note: gzip -d does the same as gunzip.

4. bzip2 - Burrows-Wheeler Compression

Compress files with better ratio than gzip (slower).

# Compress file
bzip2 file.txt

# Compress keeping original
bzip2 -k file.txt

# Decompress
bzip2 -d file.txt.bz2

# Set compression level (1-9)
bzip2 -9 file.txt       # Maximum compression

# Test integrity
bzip2 -t file.txt.bz2

5. bunzip2 - Decompress bzip2

Decompress files compressed with bzip2.

# Decompress file
bunzip2 file.txt.bz2

# Decompress keeping original
bunzip2 -k file.txt.bz2

# Test integrity
bunzip2 -t file.txt.bz2

6. xz - LZMA Compression

Compress files with best ratio (slowest but most efficient).

# Compress file
xz file.txt

# Compress keeping original
xz -k file.txt

# Decompress
xz -d file.txt.xz

# Set compression level (0-9)
xz -9 file.txt          # Maximum compression

# Test integrity
xz -t file.txt.xz

# Show compression info
xz -l file.txt.xz

7. zip - Create ZIP Archives

Create ZIP format archives (Windows compatible).

# Create zip archive
zip archive.zip file1.txt file2.txt

# Add directory recursively
zip -r archive.zip directory/

# Add files to existing archive
zip archive.zip newfile.txt

# Update only newer files
zip -u archive.zip file.txt

# Delete file from archive
zip -d archive.zip file.txt

# List contents
zipinfo archive.zip

# Encrypt with password
zip -e archive.zip file.txt

8. unzip - Extract ZIP Archives

Extract files from ZIP archives.

# Extract to current directory
unzip archive.zip

# Extract to specific directory
unzip archive.zip -d /destination/

# List contents
unzip -l archive.zip

# Extract specific files
unzip archive.zip file1.txt file2.txt

# Extract without overwrite
unzip -n archive.zip

# Test integrity
unzip -t archive.zip

# Quiet extraction
unzip -q archive.zip

9. 7z - 7-Zip Archive

High compression ratio archive format.

# Install 7zip
sudo apt install p7zip-full

# Create 7z archive
7z a archive.7z file1.txt directory/

# Extract archive
7z x archive.7z

# Extract to specific directory
7z x archive.7z -o/destination/

# List contents
7z l archive.7z

# Set compression level (0-9)
7z a -m9=deflate archive.7z file.txt

# Encrypt with password
7z a -p archive.7z file.txt

10. zcat - View Compressed Files

View compressed files without extracting.

# View gzip file content
zcat file.txt.gz

# View and pipe to less
zcat file.txt.gz | less

# View multiple files
zcat file1.gz file2.gz

# Similar commands
bzcat file.txt.bz2      # For bzip2
xzcat file.txt.xz       # For xz

🚀 AI Use Case: Split Compression for Large Model Weight Files

When working with large AI model files (GGUF, AWQ, Safetensors), split compression is essential for sharing models, uploading to cloud storage with file size limits, or distributing models across multiple storage devices.

# Compress and split a large GGUF model (e.g., Llama-3-70B.gguf ~40GB)
# Split into 4GB chunks for easy upload

# Step 1: Compress the model file
xz -9 -k llama-3-70b.gguf  # Creates llama-3-70b.gguf.xz

# Step 2: Split compressed file into 4GB chunks
split -b 4G llama-3-70b.gguf.xz llama-3-70b.gguf.xz.part.

# Result: llama-3-70b.gguf.xz.part.aa, .ab, .ac, etc.

# Step 3: List all parts
ls -lh llama-3-70b.gguf.xz.part.*

# Step 4: Reconstruct the file
cat llama-3-70b.gguf.xz.part.* > llama-3-70b.gguf.xz

# Step 5: Decompress
xz -d llama-3-70b.gguf.xz

Alternative: Direct split without compression (faster but larger):

# Split large model file directly (e.g., for USB transfer)
split -b 4G llama-3-70b.gguf llama-3-70b.gguf.part.

# Reconstruct on destination
cat llama-3-70b.gguf.part.* > llama-3-70b.gguf

# Verify integrity with checksum
sha256sum llama-3-70b.gguf > llama-3-70b.gguf.sha256
sha256sum -c llama-3-70b.gguf.sha256

Practical scenario: Sharing a 35GB Qwen2.5-72B-Instruct.GGUF via cloud storage with 5GB file limits:

# Compress with maximum compression
xz -9 -k qwen2.5-72b.gguf  # May reduce to ~25GB

# Split into 4.5GB chunks (safe margin)
split -b 4500M qwen2.5-72b.gguf.xz qwen2.5-72b.part.

# Upload all parts to cloud storage
# Share download link with users

# Users download and reconstruct:
cat qwen2.5-72b.part.* > qwen2.5-72b.gguf.xz
xz -d qwen2.5-72b.gguf.xz

# Verify model integrity
sha256sum qwen2.5-72b.gguf

Tip: For AWQ/Safetensors models with multiple files, use tar first:

# Archive model directory
tar -cJvf model-70b.tar.xz models/llama-3-70b/

# Split for transfer
split -b 4G model-70b.tar.xz model-70b.tar.xz.part.

# Reconstruct and extract
cat model-70b.tar.xz.part.* > model-70b.tar.xz
tar -xJvf model-70b.tar.xz
