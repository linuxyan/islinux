📝 Essential Linux Text Processing Commands

10 min read

Introduction

Text processing is a fundamental skill in Linux, essential for log analysis, data manipulation, configuration management, and automation. This guide covers 10 powerful commands for processing and manipulating text files.

1. grep - Pattern Search

Search for patterns in files using regular expressions.

# Basic search
grep "error" file.txt

# Case-insensitive search
grep -i "error" file.txt

# Invert match
grep -v "debug" file.txt

# Show line numbers
grep -n "error" file.txt

# Count matches
grep -c "error" file.txt

# Show context (3 lines)
grep -C 3 "error" file.txt

# Recursive search
grep -r "function" /home/user/

# Show only filenames
grep -l "error" *.log

# Extended regex
grep -E "error|warning" file.txt

2. sed - Stream Editor

Edit and transform text streams.

# Replace first occurrence
sed 's/old/new/' file.txt

# Replace all occurrences
sed 's/old/new/g' file.txt

# Replace from specific line
sed '2s/old/new/' file.txt

# Delete lines
sed '5d' file.txt
sed '1,10d' file.txt

# Print specific lines
sed -n '5p' file.txt

# Insert before line
sed '5i\New line' file.txt

# Edit file in place
sed -i 's/old/new/g' file.txt

# Create backup while editing
sed -i.bak 's/old/new/g' file.txt

3. awk - Pattern Scanning and Processing

Powerful text processing and data extraction.

# Print first field
awk '{print $1}' file.txt

# Print specific fields
awk '{print $1, $3}' file.txt

# Print last field
awk '{print $NF}' file.txt

# Use custom delimiter
awk -F: '{print $1}' /etc/passwd

# Conditional printing
awk '$1 > 100' file.txt

# Arithmetic operations
awk '{sum += $1} END {print sum}' file.txt
awk '{avg += $1} END {print avg/NR}' file.txt

# Format output
awk '{printf "%-10s %5d\n", $1, $2}' file.txt

4. cut - Remove Sections

Extract sections from each line of files.

# Extract first field (tab delimited)
cut -f1 file.txt

# Extract specific fields
cut -f1,3,5 file.txt

# Use custom delimiter
cut -d: -f1 /etc/passwd
cut -d, -f1,3 data.csv

# Extract characters
cut -c1-5 file.txt

# Extract bytes
cut -b1-10 file.txt

5. sort - Sort Lines

Sort lines of text files.

# Basic sort
sort file.txt

# Reverse sort
sort -r file.txt

# Numeric sort
sort -n file.txt

# Human-readable numeric sort
sort -h file.txt

# Sort by specific field
sort -t: -k2 /etc/passwd

# Remove duplicates
sort -u file.txt

# Case-insensitive sort
sort -f file.txt

# Randomize order
sort -R file.txt

6. uniq - Filter Duplicates

Remove or report duplicate lines (requires sorted input).

# Remove adjacent duplicates
uniq file.txt

# Remove duplicates (with sorted)
sort file.txt | uniq

# Count occurrences
sort file.txt | uniq -c

# Show only duplicates
sort file.txt | uniq -d

# Show only unique lines
sort file.txt | uniq -u

# Case-insensitive
sort file.txt | uniq -i

7. tr - Translate Characters

Translate or delete characters.

# Convert lowercase to uppercase
echo "hello" | tr 'a-z' 'A-Z'

# Convert uppercase to lowercase
echo "HELLO" | tr 'A-Z' 'a-z'

# Delete characters
echo "hello123" | tr -d '0-9'

# Squeeze repeated characters
echo "aaabbbccc" | tr -s 'a-z'

# Replace characters
echo "hello world" | tr ' ' '_'

# Convert tabs to spaces
tr '\t' ' ' < file.txt

8. wc - Word Count

Count lines, words, and characters.

# Count lines, words, bytes
wc file.txt

# Count lines only
wc -l file.txt

# Count words only
wc -w file.txt

# Count characters only
wc -m file.txt

# Count bytes only
wc -c file.txt

# Combined with other commands
grep -r "TODO" . | wc -l

9. head/tail - First/Last Lines

View beginning or end of files.

# First 10 lines (default)
head file.txt

# First 20 lines
head -n 20 file.txt

# Last 10 lines (default)
tail file.txt

# Last 50 lines
tail -n 50 file.txt

# Follow file (log monitoring)
tail -f /var/log/syslog

# Follow with retry
tail -F /var/log/application.log

# Multiple files
tail file1.txt file2.txt

10. diff/patch - Compare and Apply

Compare files and apply patches.

# Compare two files
diff file1.txt file2.txt

# Unified format
diff -u file1.txt file2.txt

# Recursive directory comparison
diff -r dir1/ dir2/

# Ignore whitespace
diff -w file1.txt file2.txt

# Side-by-side comparison
diff -y file1.txt file2.txt

# Create patch file
diff -u original.txt modified.txt > changes.patch

# Apply patch
patch original.txt < changes.patch

# Reverse patch
patch -R < changes.patch

Conclusion

These 10 text processing commands enable you to search and filter text patterns, transform and edit text streams, extract specific data fields, sort and remove duplicates, compare files and apply patches, and analyze log files efficiently.
