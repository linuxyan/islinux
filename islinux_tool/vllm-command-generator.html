📚 Deployment FAQ

Q: What are the best Ollama environment variables for Linux in 2026?

For peak performance, always set OLLAMA_NUM_PARALLEL to match your CPU cores and ensure OLLAMA_MAX_LOADED_MODELS fits within your VRAM budget calculated by our VRAM Predictor.

Q: How to fix vLLM Out-of-Memory (OOM) errors?

Most OOM issues are caused by excessive gpu_memory_utilization. For 12GB cards, keep it at 0.9. For production workloads, consider using AWQ or GPTQ 4-bit quantization.

Q: How to choose between Native Python and Docker deployment?

Use Docker for production environments with consistent dependencies and isolation. Use Native Python for development, debugging, or when you need direct access to system libraries and CUDA tools.
