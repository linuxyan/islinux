How to Calculate TPS from GPU TFLOPS?

The relationship between GPU TFLOPS and Tokens Per Second (TPS) is governed by Transformer physics. For every token generated, each parameter must perform approximately 2 floating-point operations (FLOPs).

The Master Formula:

TPS = (GPU_TFLOPS × 1000 × MFU) / (2 × Parameters_B)

*MFU (Model FLOPs Utilization) represents real-world efficiency, typically 0.55 for optimized stacks like vLLM.

🔍 Why Real-World Speed May Vary

While this calculator provides a compute-bound estimate, actual results depend on:

  • Memory Bandwidth: For batch size = 1, GPU waits for data from VRAM (GB/s matters more than TFLOPS)
  • KV Cache Overhead: Longer conversations consume more VRAM and compute
  • Kernel Optimization: Tuned Linux kernel can improve MFU by up to 15%
